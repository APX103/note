Matrix Chat Agent System

一个极简但完整的多 Agent 协作演示系统。基于 Synapse Homeserver + Element Web， 支持人类与 Agent、Agent 与 Agent 在 Matrix 聊天室中实时对话。

🔧

Synapse

Matrix 官方 Homeserver，负责用户管理、消息路由、房间管理

💬

Element Web

Matrix 官方 Web 客户端，人类通过浏览器与 Agent 交互

🤖

Bot (Python)

基于 matrix-nio 的 Agent，支持自动注册、历史上下文、防循环

🧠

LLM API

兼容 OpenAI API 的大模型接口，负责生成对话回复

消息传递链路

一条消息从人类发出到 Bot 回复的完整链路

sequenceDiagram participant U as Alice participant E as Element Web participant S as Synapse participant B as Bot participant L as LLM U->>E: 输入：@bot 你好 E->>S: PUT /send/m.room.message S->>S: 持久化消息 S-->>B: /sync 长轮询推送 B->>B: is_mention 检测 B->>B: 加载房间历史（20条） B->>L: Chat Completion（含上下文） L-->>B: 生成回复 B->>B: clean_reply 清洗 B->>S: PUT /send/m.room.message S->>S: 持久化消息 S-->>E: /sync 推送 E-->>U: 显示 Bot 回复

🔍 Mention 检测逻辑

explicit_mention = (
    USER_ID in body          # @bot:localhost
    or f"@{USERNAME}" in body # @bot
)

fuzzy_match = USERNAME.lower() in body.lower()

# 群聊：只响应明确 @mention
# 私聊：响应模糊匹配
is_mention = explicit_mention or (is_dm and fuzzy_match)

🔄 防循环机制

# 冷却时间（同一房间10秒内只回复一次）
if now - last < COOLDOWN_SECONDS:
    跳过

# Bot 对话深度限制
bot_chain = 0
for msg in reversed(history):
    if is_bot(msg.sender):
        bot_chain += 1
    else:
        break

if bot_chain >= MAX_BOT_CHAIN:  # 默认20
    跳过

Bot-Bot 对话链路

两个 Agent 在聊天室中互相 @mention 触发对话

sequenceDiagram participant U as Alice participant S as Synapse participant B1 as Bot1 (@bot) participant B2 as Bot2 (@bot2) participant L as LLM U->>S: @bot 你跟 bot2 聊3轮 S-->>B1: 推送消息 B1->>L: 生成回复 L-->>B1: @bot2:localhost 你好！ B1->>S: 发送回复 S-->>B2: 推送 @mention B2->>B2: is_mention = True B2->>L: 生成回复（含历史） L-->>B2: @bot:localhost 你好！ B2->>S: 发送回复 S-->>B1: 推送 @mention B1->>B1: 检查 bot_chain=2 B1->>L: 生成第3轮回复 L-->>B1: @bot2:localhost 再见～ B1->>S: 发送 S-->>B2: 推送 B2->>B2: bot_chain=3 >= MAX B2->>B2: 静默跳过，对话终止

上下文传递格式

Bot 调用 LLM 时，会把整个房间历史包装成对话格式传入

  {role: "system", content: "你是 @bot:localhost..."},
  {role: "user",     content: "alice: @bot 你好"},
  {role: "assistant", content: "bot: 你好 Alice！"},
  {role: "user",     content: "bot2: @bot 你好！很高兴见到你"},
  {role: "user",     content: "bob: @bot 你跟 bot2 聊3轮"},
]

项目结构

代码组织与关键文件说明

matrix-chat/
├── docker-compose.yml          # Synapse + Element Web
├── synapse-data/               # Homeserver 数据（SQLite）
├── element-web-config/
│   └── config.json             # Element 默认指向本地 Synapse
├── bot/
│   ├── bot.py                  # Agent 主程序（matrix-nio + OpenAI）
│   ├── config.yaml             # Bot1 配置（@bot:localhost）
│   ├── config2.yaml            # Bot2 配置（@bot2:localhost）
│   ├── requirements.txt        # Python 依赖
│   ├── run-bot1.sh             # 启动脚本
│   ├── run-bot2.sh
│   └── .venv/                  # uv 虚拟环境
└── index.html                  # 本介绍页面

bot.py 核心模块

  • • login_or_register - 自动注册/登录
  • • load_room_history - 拉取房间历史
  • • on_message - 消息事件处理
  • • ask_llm - 调用大模型生成回复
  • • clean_reply - 清洗 LLM 输出

配置文件

  • • matrix.* - Homeserver / 账号
  • • llm.* - API 地址 / Key / 模型
  • • bot.mode - echo / llm
  • • bot.max_bot_chain - 最大对话轮数
  • • bot.cooldown_seconds - 冷却时间

关键技术选型

  • • Synapse - Matrix 官方 Homeserver
  • • matrix-nio - Python Matrix 客户端
  • • openai - OpenAI API 兼容客户端
  • • PyYAML - 配置文件解析
  • • uv - Python 包管理
